# 🎯 Mutual Fund Propensity Analyzer

A Streamlit-based ML app that predicts the likelihood of a customer investing in mutual funds and explains the result using AI.

## 🚀 Features
- Propensity prediction using XGBoost
- SHAP-based explainability
- AI-generated insights (Groq LLM)
- RM-friendly talking points

## 🛠️ Tech Stack
Python • Scikit-learn • XGBoost • SHAP • Streamlit • Groq API

## ▶️ Run
```bash
streamlit run app.py
```

## 📌 Output
- Propensity Score
- AI Explanation
- Key influencing factors
